# Character Bios Website

A single-page website for character bios with dropdown navigation, search functionality, and thematic SVG decorations.

## Features

- Dropdown menus for each character type
- Info and Backstory sections for each character
- Search and filter characters
- Gothic font styling
- SVG decorations for each species
- Responsive design

## Characters Included

- Human
- Wolf Hybrid
- Fox Hybrid
- Dragon Hybrid
- Dog Hybrid
- Cat Hybrid
- Moth Hybrid
- Harpy Hybrid
- Angel
- Fallen Angel
- Demon
- Vampire
- Werewolf
- Zombie
- Naga

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Cinzel)

## Setup

1. Clone the repository
2. Open `index.html` in a web browser

## Deployment

This site is designed to be deployed on GitHub Pages or any static hosting service.
